# v4.1 Demo Runbook (CLI + UI)

Dokumen ini untuk menyiapkan demo singkat v4.1, termasuk generator transcript CLI dan pipeline pembuatan aset GIF/MP4.

## 0) Setup

```bash
npm install
npm run build
```

## 1) Capture CLI transcript

```bash
npm run demo:v41:cli
```

Output transcript disimpan ke:
- `artifacts/demo/v41-cli-demo.txt`

Jika ingin transcript dari benchmark fresh (rebuild + benchmark):

```bash
npm run demo:v41:cli:fresh
```

## 2) Demo UI (Next.js example existing)

Gunakan app existing berikut:

```bash
cd examples/standar-config-next-js-app
npm run dev
```

Flow demo UI (30–60 detik):
1. Tunjukkan heading backlog demo.
2. Klik **Add to cart** (state counter berubah).
3. Klik **Switch theme** (live token/theme switching).
4. Ubah lebar viewport untuk menunjukkan container-query grid.

## 3) Generate GIF/MP4 assets

Siapkan tool berikut (opsional tapi direkomendasikan):
- `terminalizer` (rekam terminal),
- `ffmpeg` (konversi video/GIF),
- `gifsicle` (optimasi GIF).

Script helper:

```bash
bash scripts/demo/generate-demo-assets.sh
```

Script akan:
- membuat `docs/assets/demo-cli.gif` dari recording terminal,
- mencoba optimasi ke `docs/assets/demo-cli-optimized.gif` jika `gifsicle` tersedia,
- memberi instruksi perekaman UI manual (save sebagai `docs/assets/demo-ui.mp4`),
- jika `demo-ui.mp4` tersedia + `ffmpeg` ada, membuat:
  - `docs/assets/demo-ui.gif`,
  - `docs/assets/demo-full.mp4` (gabungan CLI + UI).

## 4) Asset status

- CLI transcript: ✅ ada (`artifacts/demo/v41-cli-demo.txt`)
- GIF/video final: CLI transcript + runbook tersedia (GIF recording adalah manual step post-release)
