# Announcing tailwind-styled-v4 Tailwind v4 Upgrade Path

We have rolled out a staged upgrade path for Tailwind v4:

- Core parser/theme/merge/styled primitives
- Scanner + Engine architecture
- CLI migration utilities (`init`, `scan`, `migrate`, `--wizard`)
- Vite integration with scanner/engine build hooks

Next milestones focus on:

- native parser optimization (Rust + fallback)
- additional adapter hardening
- release stabilization and broader compatibility matrix
