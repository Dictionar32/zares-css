# Code of Conduct

Project ini berkomitmen menyediakan lingkungan kolaborasi yang aman dan inklusif.

## Prinsip

- Saling menghormati.
- Kritik fokus ke ide/kode, bukan personal.
- Tidak mentolerir pelecehan dalam bentuk apa pun.

## Enforcement

Jika terjadi pelanggaran, laporkan lewat kanal support di `SUPPORT.md`.
Maintainer berhak mengambil tindakan yang dianggap perlu.
