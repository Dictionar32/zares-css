## Technical Architecture

### Three-Layer Stack

```
Layer 1: TypeScript Wrapper (index.ts)
  ↓ Type-safe function calls
Layer 2: NAPI Bindings (index.d.ts - auto-generated)
  ↓ JSON marshalling
Layer 3: Rust Core (napi_bridge.rs)
  ↓ High-performance implementation
Core Libraries (parser, resolver, generator)
```

### Data Flow Example

```
Input: "md:hover:bg-blue-600/50"
  ↓
parseClass() [TypeScript]
  ↓
native.parseClass() [NAPI]
  ↓ (Rust)
parse_class_impl()
  ↓
JSON: '{"variants":["md","hover"],"prefix":"bg",...}'
  ↓
JSON.parse() [TypeScript]
  ↓
ParsedClassResult { variants, prefix, value, modifier }
```

## Usage Examples

### Basic Parsing
```typescript
import { parseClass } from './native'

const result = parseClass("md:hover:bg-blue-600/50")
// { variants: ["md", "hover"], prefix: "bg", value: "blue-600", modifier: "50" }
```

### Theme Resolution
```typescript
import { resolveColor, resolveSpacing } from './native'

const color = resolveColor("blue-600")      // "#2563eb"
const spacing = resolveSpacing("4")         // "1rem"
```

### Single Class Compilation
```typescript
import { compileToCss } from './native'

const css = compileToCss("md:hover:bg-blue-600/50")
// "@media (min-width: 768px) { ... }"
```

### Batch Compilation
```typescript
import { compileToCssBatch } from './native'

const css = compileToCssBatch(["bg-blue-600", "text-white", "p-4"])
// ".bg-blue-600 { background-color: #2563eb; } .text-white { color: white; } ..."
```

### Type Safety
```typescript
import { compileClass, type CssRuleResult } from './native'

const rule: CssRuleResult = compileClass("text-white")
// TypeScript ensures correct types
```
