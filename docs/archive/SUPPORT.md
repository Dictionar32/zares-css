# Support

## Butuh bantuan?

- Buka issue untuk bug/feature request.
- Gunakan template issue agar cepat diproses.

## Security

Jangan laporkan celah keamanan di issue publik. Hubungi maintainer secara privat.

## Expected response

Maintainer berusaha menanggapi issue valid secepat mungkin sesuai kapasitas.
