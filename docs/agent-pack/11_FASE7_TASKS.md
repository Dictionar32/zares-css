# FASE 7 — Community & Maintenance

## Tujuan
Menyediakan standar kontribusi dan kanal support agar adopsi dan maintenance berkelanjutan.

## Task 7.1 Community files
Tambahkan file:
- `CONTRIBUTING.md`
- `CODE_OF_CONDUCT.md`
- `SUPPORT.md`

## Task 7.2 Issue templates
Tambahkan GitHub issue templates:
- bug report
- feature request
- config template

## Exit criteria Fase 7
- Kontributor baru dapat onboarding via dokumen.
- Issue masuk lebih terstruktur dan actionable.
