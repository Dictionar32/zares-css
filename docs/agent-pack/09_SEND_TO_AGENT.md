# Cara Kirim ke AI Agent

1. Kirim `01_START_HERE.md` terlebih dulu.
2. Lampirkan seluruh file `02` s.d. `08`.
3. Pakai instruksi pembuka berikut:

```text
Ikuti dokumen agent-pack secara berurutan.
Mulai dari Fase 0 Task 0.1.
Setiap task harus dilaporkan dengan format:
✅ [TASK_ID] COMPLETE
- perubahan
- validasi
- next task
Jika ada blocker, hentikan dan jelaskan blocker + opsi solusi.
```

## Jika tidak bisa upload file
Tempel dengan pemisah:

```text
=== FILE: 01_START_HERE.md ===
...
=== FILE: 02_FASE0_TASKS.md ===
...
```
